BM25 公式渲染版

1. 总公式

query q 和 topic document d 的 BM25 分数
score ( q , d ) = ∑ t ∈ q IDF ( t ) ⋅ tf ( t , d ) ⋅ ( k1 + 1 ) tf ( t , d ) + k1 ⋅ ( 1 - b + b ⋅ |d| avgdl )

2. IDF 公式

词 t 的区分度
IDF ( t ) = log ( 1 + N - df ( t ) + 0.5 df ( t ) + 0.5 )

3. 长度惩罚项

代码里的 denom_base
denom_base = k1 ⋅ ( 1 - b + b ⋅ |d| avgdl )

4. 单个词的贡献

term_score
term_score ( t , d ) = IDF ( t ) ⋅ tf ( t , d ) ⋅ ( k1 + 1 ) tf ( t , d ) + denom_base

5. 你的工程里的含义

公式符号 在你的工程里是什么意思
q 当前 specific query，例如 “What did the team discuss about the product cost?”
d 某个 topic lexical document，例如 topic 2 的 “Product cost discussion + representative turns”
t query 里的词，例如 product 或 cost
tf(t,d) 词 t 在 topic document d 里出现的次数
df(t) 有多少个 topic document 出现过词 t
N 当前 meeting 里的 topic document 总数
|d| / avgdl 当前 topic document 相对平均 topic document 的长度比例，用来做长度惩罚

为什么例子里 tf = 3？

因为假设 label_repeat = 3。例如 topic 2 的 label 是 Product cost discussion，重复 3 次后，product 和 cost 都出现 3 次，所以 tf(product, topic2)=3，tf(cost, topic2)=3。

6. 一句话

BM25 粗路由做的事情：
看 query 里的关键词在每个 topic document 里出现得多不多，
再给稀有关键词更高权重，
同时惩罚过长 topic document，
最后选分数最高的 topic。
